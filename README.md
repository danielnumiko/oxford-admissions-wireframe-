# Oxford Graduate Admissions Wireframe

Low-fidelity wireframe prototype for the Oxford Graduate Admissions update.

## Features

- **Course Page (DPhil)**: Individual projects accordion, core components, option modules, and funding section
- **Project Finder**: Global search with filters (department, research area, funded only)
- **Department Page**: Course list with featured studentships

## Design Aesthetic

- Strict low-fi "bones" aesthetic
- Colors: White (#FFFFFF), Light Gray (#F3F4F6), Black (#000000)
- No rounded corners, shadows, or gradients
- 1px solid borders on all containers

## Usage

Open `index.html` in a web browser. Use the view switcher at the top to toggle between the three views.
